# CodeAlpha_Unemployment_Analysis

CodeAlpha Data Science Internship — **Task 2: Unemployment Analysis with Python**

## Dataset (Official)

**Kaggle:** https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

Files in `data/`:
- `Unemployment in India.csv`
- `Unemployment_Rate_upto_11_2020.csv`

To re-download from Kaggle (requires [Kaggle API setup](https://www.kaggle.com/docs/api)):

```bash
pip install kaggle
python download_data.py
```

## Setup

```bash
pip install -r requirements.txt
python main.py
```
